\begin{abstract}

\noindent
\module{mod_python} を使って Apache サーバに Python を埋め込むと、
大幅に速度を改善しつつ柔軟に Web ベースアプリケーションを設計できる
ようになります。
\noindent
このドキュメントは\module{mod_python} の網羅的なリファレンスに加え、
ユーザガイドやチュートリアルとしても使える必要かつ的確な情報源を
目指して書かれています。
